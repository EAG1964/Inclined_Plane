2. Reference System:
Axes centered on the CM. X parallel to the plane, Y perpendicular.
3. Weight Representation:
W acts vertically downwards; W = m • g = 0 N
4. Decomposing Weight:
Angle α is translated between the -Y axis and W.
Wx = W • sin(α) ; Wx = 0 N
Wy = W • cos(α) ; Wy = 0 N
5. Represent Normal Force:
N is the plane's reaction; N = Wy = 0 N
6. Applied Force:
External F in -- direction; F = 0 N
7. Friction Force:
Ff opposes motion; Ff = μ • N = 0 N

Y Axis: ΣFy = m • ay N - Wy = 0 → N = Wy X Axis: ΣFx = m • ax F + Wx - Ff = m • ax Net Force: 0 N
